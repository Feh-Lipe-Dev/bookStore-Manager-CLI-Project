# BookStore Manager CLI

## Objetivo

Este projeto tem como objetivo desenvolver uma aplicação CLI em Node.js + TypeScript para gerenciar autores, livros, clientes e empréstimos de uma livraria.

## Status atual

A estrutura inicial do projeto foi criada para dar início ao desenvolvimento.

## Próximos passos

1. Configurar a conexão com PostgreSQL.
2. Criar o script SQL do banco de dados.
3. Implementar a arquitetura em camadas.
4. Criar os menus e as operações CRUD.

## Como executar

```bash
npm install
npm run dev
```
